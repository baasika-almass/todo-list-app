# ✅ To-Do List App

A Python terminal app to manage your daily tasks.
Tasks are saved to a file so they stay even after closing!

## Features
- Add new tasks
- Mark tasks as complete
- Delete tasks
- View all tasks with status
- Saves tasks automatically to a JSON file

## How to Run
```bash
python todo_list.py
```
